# spacing_select

## Description
`spacing_select.c` is a Motif GUI source file, typically generated by an interface builder tool like X-Designer. Its purpose is to create and manage a dialog box titled "Inter-Row Spacings" within the "Submetrix 2000 unraveller" application. This dialog provides text input fields for users to configure critical transducer spacing parameters for both port and starboard sides, including nominal spacings and various ratio values (e.g., AB-AC, AB-AD ratios). These parameters are essential for accurate phase-to-angle conversion in phase-measuring sonar systems.
